# house-price-prediction
# 🏠 House Price Prediction App

This project is a Machine Learning web application that predicts house prices based on user inputs like BHK, total square feet, number of bathrooms, and balconies.

It is built using **Streamlit** and a **Linear Regression model** trained on housing data.

---

## 🚀 Features

* 📊 Predict house prices instantly
* 🧠 Machine Learning model using Linear Regression
* 🧹 Data preprocessing and cleaning
* 🌐 Interactive user interface with Streamlit

---

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Streamlit
* Matplotlib
* Joblib

---

## 📊 Model Details

* Algorithm: Linear Regression
* Features used:

  * Size (BHK)
  * Total Square Feet
  * Number of Bathrooms
  * Number of Balconies
* Evaluation Metric: R² Score

## ▶️ How to Run

### 1. Clone the repository

git clone https://github.com/Thatipramod/house-price-prediction
cd house-price-prediction

---

### 2. Install dependencies

pip install streamlit numpy pandas scikit-learn matplotlib joblib

---

### 3. Run the application

streamlit run app.py

---

## 💡 Usage

* Enter:

  * Number of BHK
  * Total square feet
  * Number of bathrooms
  * Number of balconies

* Click **Predict Price**

* The app will display the estimated house price
